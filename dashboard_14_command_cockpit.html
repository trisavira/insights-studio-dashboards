Role. The cockpit holds no data of its own — it is a pure aggregation layer over the thirteen operational dashboards. Its job is triage and routing: a Medical Affairs lead opens it first each morning, reads system status in ten seconds, works the attention queue top-down, and clicks through to the dashboard that owns each item. In Tableau this is a single navigation workbook whose tiles and roll-ups reference the published views; in production it would be the Studio's landing page. This completes the Insights Studio — 13 operational surfaces plus the cockpit that ties them together.
