Integrity
CARVYKTI carries an evidence-integrity overhang. The FDA opened a class-wide secondary-malignancy investigation (Jan 2024), and the first SPM case “almost certainly from insertional mutagenesis” was reported in early 2025. This is neither a classic efficacy nor a safety event — it is an evidence-integrity / causality question the architecture holds in a Compromised state (purple). ?
